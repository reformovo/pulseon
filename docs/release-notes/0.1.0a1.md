# PulseOn 0.1.0a1

This is an alpha release for validating the v1 native loop before a stable
compatibility commitment.

## Scope

- Local-first project and run tracking.
- Numeric metric logging through the Python SDK.
- Local metric queries, downsampling, summaries, and metric discovery.
- DuckLake-backed native storage with the PulseOn Parquet schema as the product
  compatibility boundary.

## Compatibility

- Python package version: `0.1.0a1`.
- Git tag: `v0.1.0a1`.
- Requires Python 3.11 or newer on CPython.
- Wheels use the CPython 3.11+ stable ABI; the release also includes an sdist
  for source builds.
- The v1 Parquet schema contract is documented in
  `docs/v1-parquet-schema-contract.md`.

## Known Limits

- This is not a stable v1 release.
- Cloud, workspace hierarchy, config/tag filtering, built-in plotting, and AI
  Native features are outside this release.
- DuckDB LTTB downsampling requires an installed extension, automatic extension
  installation, or `PULSEON_LTTB_EXTENSION_PATH`.
